Flocking Simulation

FPS: --

1500 individual boids, using the 3 classic flocking rules (Seperation, Alignment, Cohesion) + Gary Flake's 4th View Unblocking rule

Employing bin-lattice spatial subdivision for optimisation, boid simulation computed on CPU via Rust -> WebAssembly

Rust compiled to WebAssembly for simulation computation | WebGPU canvas render-shaders rendering the positional values from Rust in parallel
